# Examples

Pre-made configurations and templates for common use cases.

## Available Examples

### Configurations

- **[minimal-config.md](minimal-config.md)** - Barebones setup (check-in only, ~2 min)
- **work-focused-config.md** - Work productivity focus with project tracking
- **health-wellness-config.md** - Health and wellness focused routine
- **student-config.md** - Academic goal tracking and study planning

### Goal Templates

- **fitness-goal.md** - Template for fitness and exercise goals
- **learning-goal.md** - Template for learning and skill development
- **project-goal.md** - Template for project-based goals
- **habit-goal.md** - Template for building new habits

### Integration Examples

- **todoist-integration.sh** - Sync tasks with Todoist
- **notion-export.sh** - Export daily notes to Notion
- **slack-webhook.sh** - Send routine completion to Slack

## Usage

### Apply a Configuration

```bash
# Copy example to your setup
cp examples/minimal-config.md .claude/commands/morning-routine/main.md
```

### Use a Template

```bash
# Copy goal template
cp examples/fitness-goal.md Goals/Get-Fit-2026.md

# Edit with your details
nano Goals/Get-Fit-2026.md
```
